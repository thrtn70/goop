What it does

  • Download from anywhere

    Paste a URL — Goop probes formats, you pick, it downloads. yt-dlp and gallery-dl bundled, so video sites, audio platforms, and image hosts all work out of the box.

  • Convert anything to anything

    Drop video, audio, or images in any common format. Smart targets, batch jobs, and an output folder of your choice. Drop a PDF too — merge, split, or compress in the same window.

  • Shrink files without thinking

    A single quality slider, or hit a target file size. Works on video, audio, images, and PDFs. Goop warns before it would trash quality.

  • Merge, split, compress PDFs

    Drop PDFs to merge in any order, split by ranges (1-3, 7-10), or compress with three quality presets — Screen, Ebook, Printer.

Power features

  • Live queue

    Per-job progress, speed, and ETA. Drag to reorder, cancel, or batch-cancel.

  • History with grid view

    Searchable, filterable. Lazy-loaded thumbnails. Multi-select for batch actions.

  • Hardware-accelerated encoding

    VideoToolbox on macOS; NVENC, QSV, and AMF on Windows. Auto-detected.

  • Pause and resume

    Freeze long encodes mid-flight. Resume after lunch — or after a restart.

  • Command palette

    ⌘K on macOS, Ctrl+K on Windows. Fuzzy search every action.

  • Cookies from browser

    Reach age-gated and members-only content using your existing browser session.

  • Keyboard-driven

    ⌘1–⌘5 for pages, ⌘N for URL, ⌘O for files.

  • Accessible by default

    WCAG 2.1 AA: focus rings, keyboard navigation, screen reader support, reduced motion.

A look inside

Up and running in 30 seconds

  1. Download the right installer.

    macOS (Apple Silicon, .dmg) or Windows (.msi) .

  2. Install.

    macOS: drag Goop.app to Applications. If Gatekeeper blocks it on first launch, run this once: 

    sudo xattr -cr /Applications/Goop.app

    Windows: run the .msi. If SmartScreen warns, click More info, then Run anyway. The app isn't code-signed yet — that's on the roadmap.

  3. Use it.

    Paste a URL, drop a file, or hit ⌘K / Ctrl+K to open the command palette. That's the whole tutorial.

Pro tip — Cookies from your browser

For age-gated, members-only, or paywalled content, Goop can read your existing browser session. Open Settings → Cookies and pick Chrome, Firefox, Safari, or Edge.

Cookies stay on your machine. Goop never uploads them, and there's no telemetry to send them anywhere.

Questions

Is Goop really free and open-source?

Yes. MIT licensed. Source on GitHub. No paid tier, no premium upsell.

Why is macOS blocking the app on first launch?

Goop isn't code-signed by an Apple Developer account yet. After installing, run this once in Terminal to clear the quarantine flag: 

sudo xattr -cr /Applications/Goop.app

That's a standard step for unsigned macOS apps; signing is on the roadmap.

Why is Windows SmartScreen warning me?

Same reason — the installer isn't signed yet. On the SmartScreen dialog, click More info, then Run anyway.

A site stopped working. I'm getting 403s.

Sites change frequently and yt-dlp ships fixes constantly. Goop fetches the latest yt-dlp automatically. If a site still misbehaves, the matching fix usually lands in yt-dlp within hours — file an issue if it persists.

Where do my downloads go?

Wherever you set it in Settings → Download folder. From History, click any item to reveal it in Finder or Explorer.

Does Goop send any of my data anywhere?

No. There's no telemetry, no accounts, no cloud. Every job runs on your machine. The only network traffic is the actual downloads you ask for, plus a release-check ping to GitHub.
